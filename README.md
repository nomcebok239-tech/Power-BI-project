# Power BI Sales Dashboard

## Project Overview
This project presents an interactive Power BI dashboard created to analyze customer and sales data. The dashboard provides visual insights that support data-driven decision-making.

## Tools Used
- Power BI
- Power Query
- Data Modeling

## Dashboard Features
- KPI Cards
- Interactive Filters and Slicers
- Bar Charts
- Pie Charts
- Trend Analysis Visuals

## Key Insights
- Customer purchasing trends were identified across different demographics
- Income levels influenced purchasing behavior
- Certain customer groups showed higher sales activity

## Skills Demonstrated
- Data Visualization
- Dashboard Development
- Data Transformation
- Interactive Reporting
- Business Intelligence
